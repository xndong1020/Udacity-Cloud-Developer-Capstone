### Udacity Inventory Management App

#### How to run

**back-end (serverless restful api)**

```sh
cd backend
npm i

# for unit tests
npm test

# for deployment
sls deploy -v
```

**frontend (Vue.js)**

```sh
cd client
npm i

# for linting
npm run lint

# for unit tests
npm test

# for local development
npm start
```

**Demo website** is [here](http://my-cloud-dev-s3.s3-website-us-west-2.amazonaws.com/)
